Designing an early-stage fixed‑wing control prototype

Design narrative (interview-ready)
Problem definition
Build a deterministic 6‑DOF simulator with sensor realism, robust autopilot behavior under disturbances, and defensible validation artifacts (plots + metrics).
Modeling assumptions
  • Rigid body 6‑DOF with Euler angles (Phase 1–2)
  • Coefficient-based aerodynamics (Phase 2)
  • Wind/turbulence as bandwidth-limited stochastic process (Phase 3/6)
Control law design
  • Baseline: PID cascades (airspeed, altitude→pitch, heading→bank)
  • Advanced hooks: numerical linearization + LQR utilities + gain scheduling scaffolding
Validation results
  • Altitude step response plot
  • Open-loop eigenvalues plot
